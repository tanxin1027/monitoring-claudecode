# MySQL 监控前端页面说明

## 页面概述
MySQL 监控页面提供了对 MySQL 实例全方位的性能监控，包含连接、查询、事务、缓冲池、锁、临时表、排序、复制等关键指标。

## 页面功能特性

### 1. 实时数据展示
- 每30秒自动刷新监控数据
- 支持手动刷新
- 实例选择器

### 2. 核心监控指标
#### 连接相关
- **当前连接数** - 实时连接数
- **最大连接数** - 最大允许连接数
- **连接使用率** - 当前连接数占最大连接数的百分比

#### 性能相关
- **QPS** - 每秒查询数
- **QPS 总数** - 累计查询总数
- **TPS** - 每秒事务数
- **TPS 总数** - 累计事务总数

#### 慢查询相关
- **慢查询速率** - 每秒慢查询数量
- **慢查询总数** - 累计慢查询数量

#### 缓冲池相关
- **缓冲池命中率** - 缓冲池效率指标
- **缓冲池大小** - 缓冲池总大小
- **缓冲池空闲页数** - 未使用的页数
- **缓冲池脏页数** - 待写入的页数

#### 锁相关
- **表锁等待率** - 表锁等待的百分比
- **行锁等待时间** - 行锁等待的时间
- **行锁等待次数** - 行锁等待的次数

#### 临时表和排序
- **磁盘临时表比率** - 临时表使用磁盘的比例
- **排序统计** - 各种排序操作的数量

#### 复制状态（如果适用）
- **从库 IO 状态** - 从库 IO 线程状态
- **从库 SQL 状态** - 从库 SQL 线程状态
- **主从延迟** - 落后主库的秒数

### 3. 视觉化特性
- **颜色编码** - 根据阈值显示不同颜色
- **进度条** - 性能趋势展示
- **数据卡片** - 关键指标一目了然
- **详细表格** - 完整指标数据

### 4. 颜色状态说明
- **绿色** - 正常状态
- **黄色** - 警告状态
- **红色** - 危险状态

## 性能阈值参考

| 指标 | 正常 | 警告 | 危险 |
|------|------|------|------|
| 缓冲池命中率 | > 95% | 85-95% | < 85% |
| 连接使用率 | < 70% | 70-85% | > 85% |
| 磁盘临时表比率 | < 25% | 25-50% | > 50% |
| 表锁等待率 | < 5% | 5-20% | > 20% |
| 慢查询速率 | < 1/秒 | 1-5/秒 | > 5/秒 |
| 主从延迟 | < 10秒 | 10-60秒 | > 60秒 |
